Vragen

Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?

Antwoord: Als je bij de confirm functie op 'ok' klikt dan is de return value true en bij 'annuleren' is return value false.

Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op 'ok' klikt?

Antwoord: Wanneer je tekst intypt en op ok drukt dan is de return value de tekst zelf of de defaultwaarde als je niks intypt.

Wat is de return value van de prompt functie als de gebruiker op 'cancel' klikt?

Antwoord: De return waarde zall null zijn.
